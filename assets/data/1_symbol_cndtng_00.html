เทคนิคการแก้ปัญหาเงื่อนไขสัญลักษณ์อย่างรวดเร็ว

  1. ปรับเครื่องหมายเชิงปฏิเสธที่พบ
  2. ตรวจหาเครื่องหมายสวนทางกัน ถ้าพบ สรุปว่า ไม่แน่ชัดได้เลย
  3. ต้องทำข้างหนึ่งของเงื่อนไขให้เหมือนข้างหนึ่งของข้อสรุป ก่อนที่จะนำอีกข้างมาเทียบ เสมอ
  4. การตัดสิน ให้ดูว่า ข้อสรุปครอบคลุมเงื่อนไขที่พิสูจน์ได้หรือไม่
  5. ดูภาพรวมแนวทางแก้ปัญหา ถ้ายากให้ข้ามไปก่อน


หลักการใหญ่ ๆ ของ การแก้ปัญหาเงื่อนไขสัญลักษณ์ โดยทั่วไป ก็เหมือนกับการแก้สมการ เช่น ย้ายข้างได้ กลับเศษเป็นส่วนได้ เป็นต้น แต่มีหลักการหรือขั้นตอน ดังนี้

3 ขั้นตอน การแก้ปัญหาโจทย์ เงื่อนไขสัญลักษณ์
  1. เตรียมเงื่อนไข-ข้อสรุป
    • เปลี่ยนเครื่องหมาย
      ไม่มากกว่า (≯) เป็น น้อยกว่าหรือเท่ากับ (≤)
      ไม่น้อยกว่า(≮) เป็น มากกว่าหรือเท่ากับ (≥)
      ไม่มากกว่าหรือเท่ากับ (≱) เป็น น้อยกว่า(<)
      ไม่น้อยกว่าหรือเท่ากับ(≰) เป็น มากกว่า (>)
    • ถ้าเงื่อนไขเป็นเศษส่วน ควรทำให้อยู่ในรูปจำนวนเต็ม โดย
      ถ้าส่วนเป็นตัวอักษร ให้กลับเศษเป็นส่วนทั้ง 2 ข้าง และ เปลี่ยนเครื่องหมายเป็นตรงข้าม เช่น
      1/A > 1/B เป็น A < B
      หาตัวมาคูณเพื่อให้ส่วนหมดไป เช่น
      A/2 > B/4   เอา 4 คูณทั้งสองข้าง ได้เป็น 2A > B
    • หรืออาจจะติดส่วนเอาไว้ก็ได้ เผื่อว่า มีอะไรมาตัดเพื่อกำจัดส่วนได้ภายหลัง แต่อย่าลืมทำทั้งสองข้างของอสมการ
    • บางครั้ง ข้อสรุป เป็นเศษส่วน หรือสามารถตัดทอนลงได้ ก็ควรทำ เช่น 1/A > 1/B เป็น A < B

  2. พิสูจน์เงื่อนไขเพื่อไปเปรียบเทียบกับข้อสรุป

    ในแต่ละข้อสรุป ให้ตรวจสอบว่ามีตัวเชื่อมมากกว่า 1 ตัวเชื่อมหรือไม่ ถ้ามีตัวเชื่อม มากกว่า 1 ตัวเชื่อมให้ตรวจสอบทุกตัวเชื่อม ถ้าความสัมพันธ์ผ่านตัวเชื่อมหนึ่ง มีผลไม่แน่ชัด แต่ ถ้าผ่านอีกตัวเชื่อมหนึ่ง มีผลเป็น มากกว่า ให้ถือว่า ตัวหนึ่ง มีค่ามากกว่า อีกตัวหนึ่ง เช่น
    เงื่อนไขที่ 1: A > B > C < D
    เงื่อนไขที่ 2: B > E > D > F
    ข้อสรุป: A > D

    จาก A ไป D มี 2 เส้นทาง คือ ผ่าน B, C และผ่านตัวเชื่อม D
    จาก A ไป D ผ่าน B, C มีผลเป็นไม่แน่ชัด เพราะมีเครื่องหมายสวนทางกัน
    จาก A ไป D ผ่าน B (จาก เงื่อนไขที่ 1 ไปเงื่อนไขที่ 2) มีผลเป็น มากกว่า
    กรณีนี้ ให้สรุปว่า A > D
    ข้อสรุปของโจทย์ เป็นจริง

    หรือ ถ้าผลการใช้เส้นทางหนึ่ง ขัดกับผลของอีกเส้นทางหนึ่ง ก็ให้สรุปว่า ไม่แน่ชัด เพราะอาจจะมีผลได้มากกว่า 1 อย่าง

    ในการพิสูจน์เงื่อนไขมีหลักการสำคัญ คือ


    ต้องทำให้ด้านใดด้านหนึ่งของเงื่อนไข ให้เท่ากับ ด้านใดด้านหนึ่งของข้อสรุปเสียก่อน

    จากนั้นจึงนำอีกด้านหนึ่งของข้อสรุปมาเทียบ

    อย่าทำ 2 ด้านพร้อมกัน เพราะใช้ได้ไม่ทุกกรณี


    ตัวอย่าง

    เงื่อนไขที่ 1: 2A > B > 2C
    เงื่อนไขที่ 2: K < C > 2G

    ข้อสรุป: G < A

    เราต้องพิสูจน์เงื่อนไขว่า เป็นไปตามข้อสรุปหรือไม่ โดย
    1. การทำให้ด้านใดด้านหนึ่งของเงื่อนไข มีค่าเท่ากับด้านใดด้านหนึ่งของข้อสรุป
      • ถ้ามี 2 เงื่อนไข หรือมี เงื่อนไข 2 บรรทัด ให้หาตัวเชื่อม ถ้าตัวเชื่อมไม่เท่ากัน ต้องทำให้เท่ากันเสียก่อน โดยการหาตัวคูณมาคูณตลอดทั้งแถว เช่น

        จากตัวอย่างข้างบน ควรทำเงื่อนไขที่ 2 ให้ เป็น 2C เพื่อให้ใช้เป็นตัวเชื่อมได้ โดยเอา 2 คูณตลอด ได้เป็น

        2K < 2C > 4G

      • จัดเรียงเงื่อนไขใหม่ ตามข้อสรุป
        จากตัวอย่างข้างบน ให้เขียนจาก G ไป A คือ

        เงื่อนไข ที่ 1: 2A > B > 2C
        เงื่อนไข ที่ 2: 2K < 2C > 4G

        จาก G ไป A คือ

        4G < 2C < B < 2A
        (ระวัง เปลี่ยนเครื่องหมายให้ถูกด้วย)

        หมายเหตุ
        ถ้าสังเกตว่ามีเครื่องหมายสวนทางกัน เช่น
        4G < 2C > B < 2A
        แสดงว่า เราไม่สามารถสรุป ระหว่าง G กับ A ได้ ว่า อะไร มากกว่า หรือน้อยกว่ากัน
        แต่ ต้องระวังไว้หน่อยหนึ่ง คือ ให้ดูว่า เรามีเส้นทางอื่นหรือไม่ เพราะบางครั้ง อาจจะสรุปได้ ก็ได้

      • เราเรียงลำดับได้ตามข้อสรุปแล้ว แต่ยังไม่ได้ตัวที่เท่ากับข้อสรุป ต่อไปเราต้องหาอะไร มา บวก ลบ คูณ หาร ตลอดแถว เพื่อให้ได้ ค่าตามด้านใดด้านหนึ่งของข้อสรุป เช่น
        จากตัวอย่าง ข้อสรุปต้องการ A แต่เรามี 2A เราต้องเอา 2 มาหารตลอด เพื่อให้ได้ ตามข้อสรุป
        4G/2 < 2C/2 < B/2 < 2A/2
        2G < C < B/2 < A
      • ยุบเครื่องหมาย
        จากตัวอย่าง จะเห็นว่า จาก G ไป A มีเครื่องหมาย < และตัว C และ B คั่น เราสามารถยุบเครื่องหมาย และข้ามตัวที่อยู่ระหว่างทางออกไปได้

        หลักการยุบรวม
        การยุบเครื่องหมาย ให้ยุบเครื่องหมายที่สำคัญน้อยกว่าออกไป คือ
        เครื่องหมาย > และ < สำคัญมากที่สุด
        เครื่องหมาย ≥ และ ≤ สำคัญรองลงมา
        เครื่องหมาย = สำคัญลำดับที่ 3 หรือ ข้ามไปได้เลยเพราะเท่ากัน

        จากตัวอย่าง ยุบแล้วได้เป็น
        2G < A

        ตัวอย่างอื่น ๆ

        A ≥ B = C > D
        ระหว่าง A และ D เครื่องหมาย > สำคัญมากสุด ยุบแล้ว ได้เป็น
        A > D
        ระหว่าง A และ C เครื่องหมาย ≥ สำคัญมากกว่า เครื่องหมาย = ยุบแล้ว ได้เป็น
        A ≥ C

        A = X > Y ≱ B
        เครื่องหมาย > ชนะทุกเครื่องหมาย ตัดออกแล้ว ได้เป็น
        A > B

        A ≱ X = Y > B
        ตัดเครื่องหมาย ได้เป็น
        A > B

        A = X < Y ≰ B
        เครื่องหมาย < ชนะทุกเครื่องหมาย ตัดออกแล้ว ได้เป็น
        A < B

        A < X ≰ Y < B
        ตัดเครื่องหมาย ได้เป็น
        A < B

    2. เอาอีกด้านหนึ่งของข้อสรุปมาเทียบ
      ในตัวอย่างข้อสรุปนี้ เราได้ A เหมือนข้อสรุปด้านหนึ่งแล้ว
      ต่อมาก็ต้องเอา G ตามข้อสรุปมาเทียบ
      G     2G < A
      จะเห็นว่า G น้อยกว่า 2G เขียนได้ ดังนี้
      G < 2G < A
      ยุบรวมได้เป็น
      G < A
      การพิสูจน์เงื่อนไขเสร็จเรียบร้อยแล้ว

  3. ตัดสินข้อสรุป

    การตัดสินว่าข้อสรุปเป็นจริง เป็นเท็จ หรือ สรุปได้ไม่แน่ชัด มีหลัก ดังนี้
    ให้ยึดเครื่องหมายในข้อสรุปเป็นหลัก แล้วเปรียบเทียบกับเครื่องหมายจากผลการพิสูจน์เงื่อนไข

    • กรณีข้อสรุปเป็นจริง
      • เครื่องหมายในข้อสรุป ครอบคลุม เครื่องหมายจากผลการพิสูจน์เงื่อนไข
        หรือ พูดอีกอย่างได้ว่า เครื่องหมายที่ได้จากการพิสูจน์เงื่อนไข ทุกเครื่องหมาย มึอยู่ในเครื่องหมายของข้อสรุป เช่น ข้อสรุปมีเครื่องหมายมากกว่าหรือเท่ากับ แต่การพิสูจน์เงื่อนไข มีเครื่องหมายเท่ากับเพียงเครื่องหมายเดียว เช่น
        ข้อสรุป: A ≥ B
        ผลการพิสูจน์เงื่อนไข: A = B
        หรือ
        ข้อสรุป: A ≤ B
        ผลการพิสูจน์เงื่อนไข: A < B
      • เครื่องหมายในข้อสรุป เหมือนกับ เครื่องหมายจากผลการพิสูจน์เงื่อนไข เช่น
        ข้อสรุป: A > B
        ผลการพิสูจน์เงื่อนไข: A > B
        หรือ
        ข้อสรุป: A ≤ B
        ผลการพิสูจน์เงื่อนไข: A ≤ B

    • กรณีข้อสรุปเป็นเท็จ
      กรณีข้อสรุปเป็นเท็จ คือ เครื่องหมายในข้อสรุป ไม่เหมือนกับ เครื่องหมายจากผลการพิสูจน์เงื่อนไขอย่างสิ้นเชิง คือไม่มีตัวใดเลยที่เหมือนกัน เช่น
      กรณีต่อไปนี้ เป็นเท็จทั้งสิ้น
      • ข้อสรุป: A ≥ B
        ผลการพิสูจน์เงื่อนไข: A < B
      • ข้อสรุป: A < B
        ผลการพิสูจน์เงื่อนไข: A > B

    • ข้อสรุปไม่แน่ชัด หรือสรุปไม่ได้
      • เครื่องหมายในข้อสรุป ไม่ครอบคลุม เครื่องหมายจากผลการพิสูจน์เงื่อนไขทั้งหมด
        หรือ พูดอีกอย่างได้ว่า มีเครื่องหมายที่ได้จากการพิสูจน์เงื่อนไข บางเครื่องหมาย ที่ไม่อยู่ในเครื่องหมายของข้อสรุป เช่น ข้อสรุปมีเครื่องหมายมากกว่าเพียงตัวเดียว แต่การพิสูจน์เงื่อนไข มีเครื่องหมายมากกว่าหรือเท่ากับ เครื่องหมายเท่ากับ ในผลการพิสูจน์เงื่อนไข ไม่มีอยู่ในข้อสรุป เช่น
        ข้อสรุป: A > B
        ผลการพิสูจน์เงื่อนไข: A ≥ B
        หรือ
        ข้อสรุป: A = B
        ผลการพิสูจน์เงื่อนไข: A ≤ B
      • กรณีพบว่ามีเครื่องหมายสวนทางกัน เช่น
        A > B > C < D
        การเปรียบเทียบระหว่าง A และ B คือ ไม่แน่ชัด

        ทำไมเครื่องหมายสวนทางกัน จึงสรุปไม่ได้
        ถ้า A < B > C
        ระหว่าง A กับ C จะสรุปไม่ได้ว่า มากหรือน้อยหรือเท่ากัน เพราะมีโอกาสเป็นไปได้ทั้งหมด
        เพราะทั้ง A และ C ต่างก็น้อยกว่า B เหมือนกัน
        ถ้าให้ B มีค่าเท่ากับ 100

        A ก็อาจจะเท่ากับ 50 และ C ก็อาจจะเท่ากับ 50 ก็ได้ เพราะต่างก็น้อยกว่า 100 เหมือนกัน
        กรณีนี้ สรุปได้ว่า A = C

        อีกกรณีหนึ่ง
        A = 80; C = 20 ก็ได้เหมือนกัน เพราะต่างก็น้อยกว่า 100
        กรณีนี้ สรุปได้ว่า A > C

        อีกกรณีหนึ่ง
        A = 30; C = 70 ก็ได้เหมือนกัน เพราะต่างก็น้อยกว่า 100
        กรณีนี้ สรุปได้ว่า A < C

        ดังนั้น จึงสรุปว่า ถ้าพบเครื่องหมายสวนทางกัน สรุปได้ทันทีว่า สรุปไม่ได้
        แต่มีข้อแม้นิดนึงว่า ให้ดูซิว่า มีกรณีอื่นที่ในเงื่อนไขหรือไม่ที่เราจะสามารถเปรียบเทียบระหว่าง A กับ C ได้
        ถ้าพบว่ามี และสามารถเปรียบเทียบได้ อย่างนี้ ก็จะไม่ถือว่า สรุปได้ไม่แน่ชัด เพราะสรุปได้นั่นเอง

    ให้ดูว่า แต่ละบรรทัด ตัวไหน มากหรือน้อยที่สุด แล้วเอามาเปรียบเทียบกันโดยใช้ข้อมูลจากเงื่อนไข
    ถ้าเปรียบเทียบกันได้ ให้ตอบตามผลการสรุป
    ถ้าไม่มีข้อมูลในเงื่อนไขให้เปรียบเทียบ ให้ตอบว่าไม่แน่ชัด
  • ถ้าโจทย์ให้เปรียบเทียบที่มีการบวก เช่น (A+B) > (C+D)
    หลักคิดคือ ตัวที่มากกว่ารวมกัน ย่อมน้อยกว่า ตัวที่น้อยกว่ารวมกัน
    ดังนั้น
    ถ้าพิสูจน์ว่า ทุกตัวของด้านหนึ่ง มากกว่าหรือน้อยกว่า ทุกตัวของอีกด้านหนึ่ง เราก็จะสรุปได้ว่า ด้านไหน น้อยกว่า หรือมากกว่าอีกด้านหนึ่ง
    แต่ถ้าเราพิสูจน์ได้ว่า บางตัวของด้านหนึ่งมากกว่าบางตัวของอีกด้านหนึ่ง แต่มีบางตัวกลับน้อยกว่า อย่างนี้ เราจะไม่รู้ว่า ด้านไหนมากกว่า หรือน้อยกว่าอีกด้านหนึ่ง อย่างนี้ ให้ตอบว่า ไม่แน่ชัด
    ในทางปฏิบัติ เราจึงมักจะจับคู่การพิสูจน์ เช่น
    (A+B) > (C+D)
    เราก็เปรียบเทียบ ระหว่าง A กับ C และ B กับ D หรือ ระหว่าง A กับ D และ B กับ C เป็นต้น
    ถ้าเราพิสูจน์ได้ว่า A > C และ B > D หรือ A > D และ B > C เราสรุปได้ว่า (A+B) > (C+D) เป็นจริง
    ถ้าเราพิสูจน์ได้ว่า A < C และ B < D หรือ A < D และ B < C เราสรุปได้ว่า (A+B) < (C+D) เป็นจริง
  • โจทย์ที่เป็นเศษส่วน ถ้ามีส่วนเป็นตัวอักษร ให้กลับเศษเป็นส่วน เพื่อให้ตัวอักษรอยู่ด้านบน เพื่อให้สะดวก
    ถ้าเป็นเครื่องหมายเท่ากับ ไม่ต้องเปลี่ยนเครื่องหมาย เช่น 2/F = 3/A กลับได้ เป็น F/2 = A/2
    ถ้าเป็นเครื่องหมาย น้อยกว่า หรือมากกว่า ต้องเปลี่ยนเป็นตรงข้าม เช่น
    2/F > 3/A กลับได้ เป็น F/2 < A/2
    หรือ
    2/F ≥ 3/A กลับได้ เป็น F/2 ≤ A/2 (สังเกตได้ว่า เครื่องหมายเท่ากับ ยังอยู่เหมือนเดิม)
    ถ้าจำไม่ได้ ให้ลองเปรียบเทียบกับตัวเลขดู เช่น
    1/2 = 5/10 กลับได้ เป็น 2/1 = 10/5
    คูณไขว้
    2x5 = 1x10
    10 = 10

    1/2 > 1/3 กลับได้ เป็น 2/1 < 3/1
    คูณไขว้
    2x1 < 1x3
    2 < 3 เป็นจริง กรณี มากกว่า หรือน้อยกว่า เมื่อกลับเศษเป็นส่วน ต้องเปลี่ยนเครื่องหมายเป็นตรงข้าม



คุณสมบัติของอสมการ (Properties of Inequalities)
  1. คุณสมบัติด้านการส่งผ่าน (Transitive Property)
    ถ้าเรานำค่ามาเรียงกันในทิศทางเดียวกัน เราสามารถข้ามตัวกลางได้ เช่น
    A>B>C>D เราสามารถสรุปได้ว่า A>D
  2. การบวกและลบ
    เราสามารถนำตัวเลขตัวเดียวกัน มาบวก หรือ ลบ จำนวนที่ไม่เท่ากันได้ โดยยังมีทิศทางคงเดิม เช่น
    ถ้า A > B > C แล้ว สรุปได้ว่า A+5 > B+5> C+5 หรือ
    ถ้า A+D > B+D > C+D แล้ว สรุปได้ว่า A > B > C
  3. การคูณและหาร ด้วยจำนวนเต็มบวกที่ไม่ใช่ศูนย์
    เราสามารถนำเลขจำนวนเต็มที่มีค่าเป็นบวกที่ไม่เป็นศูนย์มาคูณหรือหาร จำนวนที่ไม่เท่ากัน ได้ โดยไม่ทำให้ทิศทางการไม่เท่ากันเปลี่ยนแปลง เช่น
    ถ้าเป็นเลขจำนวนเต็มบวกที่ไม่ใช่ศูนย์ และ A > B > C แล้ว สรุปได้ว่า 5A > 5B > 5C

  4. การกลับเศษส่วน ต้องเปลี่ยนเครื่องหมายเป็นตรงข้าม เช่น มากกว่า เป็นน้อยกว่า
    ตัวอย่าง
    1/A > 3/B กลับเศษเป็นส่วน ได้เป็น A/1 < B/3
    ลองคิดถึงหลักความจริงก็ได้ เช่น
    1/5 > 1/8
    (หนึ่งส่วนในห้าส่วน ย่อมมากกว่า หนึ่งส่วนในแปดส่วน)
    เมื่อกลับเศษเป็นส่วน เราจะได้ว่า
    5/1 < 8/1 หรือ 5 น้อยกว่า 8 นั่นเอง
